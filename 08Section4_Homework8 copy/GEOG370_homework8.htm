Homework 9: Census Data Proportional Chloropleths with Various Classification Methods

Grace Parker

Below, the same dataset is shown with three different classification methods: equal interval, quantile, and natural breaks. The ratio presented in these maps is the median income of women in NY by county as a percent of the median income of men in NY by county. This ratio is best described as an association between two related attributes. This normalization follows best practices, since both attributes come from the same universe: the income of full time, year long workers age 16 or older in each county. By pulling from the same universe and by pulling data of the same type (median income), I have avoided comparing apple to oranges and ensured a meaningful map.

Equal Interval

The following map uses the equal interval classification method. Each interval has a range of 10.0 percentage points. Although a main advantage of this method is that it is easy for non-technical readers to understand, the default interval ranges do not always make this the case. For example, the default interval range for this data set was 8.2 percentage points. This was not easy to read, and many viewers might not even notice that the ranges are each equal since the range was not an integer or simple fraction. To avoid this problem, I changed the interval range to 10.0 percentage points by hand to highlight the advantages of this method.

A disadvantage of this method is that most of the values can often fall into just one or two of the categories. Indeed, in this map most values are in the lower two categories (yellow and light green). Further, there is an entire category that does not appear in the legend because no values fell in the 99.9 to 109.9 range. This method may be hiding significant differences or creating its own arbitrary differences in order to make the intervals easier to read. For this reason, this method is best for datasets that vary continuously.

Quantile

The following map uses the quantile classification method. It classifies the dataset so that the same number of counties will fall into each interval. For this set, 12-13 counties are in each interval. For this reason, the legend is a bit more difficult to read; the range begins and ends based on the data, not based on what is intuitive for the viewer. Like the equal interval method, this method can hide significant differences or create arbitrary differences. For this reason, this method is best for datasets that are equally distributed across their own range.

Natural Breaks (Adjusted)

The following map uses the natural breaks classification method, although I adjusted the numbers slightly to improve the legibility of the legend. This method is best for datasets that are unevenly distributed, but not skewed towards one end. This method is useful, since it shows meaningful differences. However, it can use wildly different interval ranges. In this map, one range is just 5 percentage points, while another is 17 points. Since this dataset is skewed, many more counties arenin the lower two intervals than in the higher intervals. Almost half of the counties on this map (30) are in the lightest green category, whereas only one county is in the darkest green interval. In addition, if I wanted to compare this dataset to the same information for five years ago, this map would not be meaningful, since the intervals are specific to this data and the older data would use different intervals.
